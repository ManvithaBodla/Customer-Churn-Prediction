# Methodology

## Step 1: Data Collection
The IBM Telco Customer Churn dataset was used.

## Step 2: Data Cleaning
Missing values were handled, and data types were corrected.

## Step 3: Exploratory Data Analysis
Various visualizations were created to understand customer behavior.

## Step 4: Feature Engineering
Categorical variables were encoded, and features were prepared for machine learning.

## Step 5: Train-Test Split
The dataset was divided into training and testing datasets.

## Step 6: Model Training
Three classification algorithms were trained:
- Logistic Regression
- Decision Tree
- Random Forest

## Step 7: Model Evaluation
Performance was measured using Accuracy, Precision, Recall, F1-score, and ROC-AUC.
